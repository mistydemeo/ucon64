Q: How do I compile flc?
A: make clean;make;make install 

Q: How do I install flc?
A: run install[_<your OS>].sh and edit $HOME/.flcrc or flc.cfg (DOS)
   (DOS version is not available yet)

Q: How do I add support for a new archive type?
A: open the config file and add the following lines
   <extension>_extract=<command>
   <extension>_test=<command>

   <extension> stands for the extension of the filename. The extension
   of test.zip would be "zip".

   <command> stands for the archiver commandline.
   
   Example: ZIP_test=zip t

   flc will add the archive name to that commandline and execute it.
