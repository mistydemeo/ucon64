Usage: flc [OPTION]... [FILE]...

  -c            also test every possible archive in DIRECTORY for errors
                  return flags: N=not checked (default), P=passed, F=failed
  --html        output as HTML document with links to the files
  --bbs         output as BBS style filelisting (default)
  -t            sort by modification time
  -X            sort alphabetical
  -S            sort by byte size
  --fr          sort reverse
  -k            show sizes in kilobytes
  --help        display this help and exit
  --version     output version information and exit
