uCON64 is the backup tool and wonderful emulator's Swiss Army knife program
It may be freely redistributed under the terms of the GNU Public License

Unlike similar tools, uCON64 is completely Open Source Software and has been ported to many platforms using code from various people

Support for almost every video game console system is continuously added and updated

Support for many different backup units to backup/restore ROM(s), and SRAM(s)
is also continuously added and updated (if their manufacturers cooperate)

Support for use and creation of RomCenter Data files (DAT)

Supports all common patch file formats like IPS (with RLE compression), APS,
BSL (Baseline Patch format), PPF (Playstation Patch File), and Game Genie

(Too) many different options for every kind of ROM or SRAM handling
Very verbose ROM information
Secure detection of bad dumps using internal checksums or DAT files
CRC32, SHA1 and MD5 calculation,
ROM comparison, modification (strip, pad, etc...)
SRAM conversion (between emulator and backup unit)
Operate as an frontend for every available emulator
Supports gzip and zip
...(to be continued)

Have a look at uCON64misc for support files like parport drivers
or at uCON64dat for DAT files generated with/for uCON64
